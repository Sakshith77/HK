📖 How to use in Blender
01
Pack your maps here
Upload AO → Red, Roughness → Green, Metallic → Blue. Click Pack. Download the PNG.
02
Import to Blender
In Shader Editor, add one Image Texture node. Load your ORM_packed.png. Set Colorspace to Non-Color.
03
Add Separate RGB node
Add a Separate Color node. Connect Image → Separate Color. R=AO, G=Roughness, B=Metallic.
04
Connect to BSDF
R → AO MixRGB factor, G → Roughness, B → Metallic on the Principled BSDF node.
05
Result
3 nodes instead of 9. Same visual result. Faster render. Less VRAM. Cleaner node graph.
